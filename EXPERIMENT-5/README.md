# Experiment 5 – Lazy Loading in React

## Aim

To implement lazy loading techniques in a Single Page Application to improve performance.

## Experiments

| # | Experiment | Description | Tech Stack |
|---|-----------|-------------|------------|
| 1 | **Component-Based Lazy Loading** | Loads components on demand using `React.lazy` and `Suspense` when a tab is clicked, reducing the initial bundle size. | React, Vite |
| 2 | **Route-Based Lazy Loading** | Loads route components only when the user navigates to that route using `React.lazy`, `Suspense`, and React Router. | React, React Router, Vite |
